Guia do Usuário - AgendaNotes

Sobre o Sistema

Bem-vindo ao AgendaNotes, um sistema de organização de Notas. Aqui está uma visão geral das principais funcionalidades:

  • Notas são organizadas em duas seções: "Próximas" e "Aguardando".
  • Próximas: Notas planejadas para o futuro com base na data fornecida.
  • Aguardando: Notas cujas datas já passaram.

Funcionalidades Principais

  • Ver Notas: Visualize suas notas organizadas.
  • Criar Nota: Adicione uma nova nota ao sistema.
  • Alterar Nota: Atualize informações de uma nota existente.
  • Excluir Nota: Remova uma nota do sistema.

Observações Importantes

Certifique-se de fornecer a data desejada para suas notas, garantindo uma organização adequada. Em caso de lentidão, pode haver atrasos na execução das operações CRUD devido ao deploy.
